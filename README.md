## Portfolio Projects

## Project 1: Pima Indian Diabetes Risk Analysis
- Cleaned clinical data by handling medically impossible zero values  
- Feature engineering: BMI categories  
- Logistic Regression model → **75.3% accuracy**  
- Visualized key clinical relationships  

**Key BME Insight**: Proper preprocessing is critical for reliable clinical algorithms.

## Project 2: Healthcare Admissions Analysis (55,500 patients)
- Calculated Length of Stay and performed detailed aggregation by condition  
- Statistical testing (One-way ANOVA on LOS: p = 0.324)  
- Enhanced visualizations with correlation heatmap  

**Key Findings**: Average stay ~15.51 days with similar patterns across major conditions.

**Technologies**: pandas, seaborn, matplotlib, scipy

## Project 3: Heart Disease UCI Risk Prediction (920 patients)
- Binary heart disease classification with proper imputation  
- Logistic Regression + Random Forest models  
- Feature importance analysis for clinical interpretability  

**Key BME Insight**: Chest pain type, age, and cholesterol are strong predictors for diagnostic tools.

**Technologies**: scikit-learn, seaborn, matplotlib

## Project 4: AI-Powered Remote Patient Monitoring Dashboard
- Simulated wearable/RPM vital signs pipeline for chronic disease flare prediction  
- Biological systems modeling (glucose-insulin index, cardio risk score)  
- Random Forest with feature importance for explainable AI  
- Interactive Plotly dashboard tailored for Pacific Islander & military populations  

**Key BME Insight**: AI-integrated remote monitoring enables proactive, personalized interventions — foundational for a scalable Hawaii-based Biomedical AI Platform serving Pacific Islander and military health needs.

**Technologies**: pandas, scikit-learn, plotly, matplotlib
